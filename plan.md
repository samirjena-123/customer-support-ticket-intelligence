# Customer Support Ticket Intelligence System

## Problem Statement
Manual handling of customer support tickets leads to delayed responses,
misclassification, and poor customer experience. This project aims to
automate ticket categorization, priority assignment, and insight extraction
using Natural Language Processing.

## Objectives
- Automatically classify support tickets into predefined categories
- Predict urgency level of tickets
- Extract useful entities from ticket text

## Input
Raw customer support ticket text

## Output
Category, priority level, and confidence score
